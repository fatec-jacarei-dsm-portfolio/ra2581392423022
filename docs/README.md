## ⚙️ Painel Admin

O portfólio possui um painel administrativo protegido por senha, acessível pelo botão **⚙ admin** no canto superior direito.

Pelo painel é possível adicionar e remover, sem editar o código:

- ➕ Novos **projetos** (com nome, descrição, tipo, tecnologias e link)
- ➕ Novas entradas de **experiência profissional ou formação**
- ➕ Novos **certificados** (com nome, emissor e ano)

> ⚠️ **Atenção:** por ser um site estático, as alterações feitas pelo painel admin não são salvas permanentemente — elas existem apenas durante a sessão atual do navegador. Para persistência real dos dados, consulte a [versão Next.js](#-versão-nextjs).
